Abstract — 摘要

We present 2DPASS, a general training framework that leverages 2D image priors to boost 3D LiDAR point cloud semantic segmentation. Unlike previous multi-modal fusion methods that require both 2D images and 3D point clouds at inference time, 2DPASS only requires 3D point clouds during testing while exploiting 2D knowledge during training. We achieve this through a multi-scale fusion-to-single knowledge distillation framework: a multi-modal fusion network learns to combine 2D and 3D features at multiple scales, and then distills this knowledge into a pure 3D network that operates independently at test time. 2DPASS achieves state-of-the-art results on SemanticKITTI (72.9% mIoU) and nuScenes (79.4% mIoU), significantly outperforming all previous LiDAR-only methods.
本文提出 2DPASS，一種通用訓練框架，利用二維影像先驗來提升三維光達點雲語義分割。不同於先前需要在推論時同時使用二維影像和三維點雲的多模態融合方法，2DPASS 在測試時僅需三維點雲，而在訓練時利用二維知識。我們透過多尺度融合至單一知識蒸餾框架實現此目標：多模態融合網路學習在多個尺度結合二維和三維特徵，然後將此知識蒸餾至在測試時獨立運作的純三維網路。2DPASS 在 SemanticKITTI（72.9% mIoU）和 nuScenes（79.4% mIoU）上達到最先進的結果，顯著超越所有先前的純光達方法。

邏輯角色「訓練時利用二維，測試時僅需三維」的設計完美解決了多模態方法的部署困難。

1. Introduction — 緒論

LiDAR-based semantic segmentation is crucial for autonomous driving. While 3D point clouds provide precise geometric information, they lack the rich texture and color cues available in 2D images. Multi-modal fusion methods address this by combining both modalities, but they require cameras to be available at inference time, increasing system complexity and cost. We propose a different strategy: use 2D images as a training-time auxiliary signal to enhance a 3D-only network, creating a model that benefits from multi-modal learning but only requires LiDAR at deployment.
基於光達的語義分割對自動駕駛至關重要。雖然三維點雲提供精確的幾何資訊，但缺乏二維影像中豐富的紋理和顏色線索。多模態融合方法透過結合兩種模態來解決此問題，但在推論時需要相機，增加了系統複雜度和成本。我們提出不同的策略：使用二維影像作為訓練時的輔助訊號來增強純三維網路，創造一個受益於多模態學習但部署時僅需光達的模型。

2. Method — 方法

2DPASS employs a two-branch architecture during training. The fusion branch takes both point clouds and projected 2D features as input, using multi-scale 2D-to-3D feature fusion at each encoder stage. The 2D features are obtained from a pre-trained image backbone and projected onto the 3D space using calibration parameters. The pure 3D branch processes only point clouds. We apply multi-scale knowledge distillation where the fusion branch serves as the teacher and the pure 3D branch as the student. The distillation loss includes feature-level distillation (L2 loss on intermediate features) and logit-level distillation (KL divergence on output predictions). At test time, only the pure 3D student branch is deployed.
2DPASS 在訓練時採用雙分支架構。融合分支同時接收點雲和投影的二維特徵，在每個編碼器階段使用多尺度二維到三維特徵融合。二維特徵由預訓練的影像骨幹提取並使用校正參數投影至三維空間。純三維分支僅處理點雲。我們應用多尺度知識蒸餾，融合分支作為教師，純三維分支作為學生。蒸餾損失包括特徵層級蒸餾（中間特徵的 L2 損失）和 logit 層級蒸餾（輸出預測的 KL 散度）。測試時僅部署純三維學生分支。

邏輯角色「教師-學生」框架是實現「訓練時融合、測試時純三維」策略的技術核心。
論證技巧 / 潛在漏洞多尺度蒸餾確保知識在不同抽象層級上的轉移。但預訓練影像骨幹的品質直接影響教師分支的表現。

3. Experiments — 實驗

On SemanticKITTI, 2DPASS achieves 72.9% mIoU, surpassing the previous best LiDAR-only method (Cylinder3D, 67.8%) by 5.1% and even outperforming several multi-modal methods that use cameras at test time. On nuScenes, 2DPASS achieves 79.4% mIoU, a 2.8% improvement over the previous state of the art. Ablation studies demonstrate that multi-scale distillation contributes +3.2% mIoU over single-scale, and the combination of feature-level and logit-level distillation yields +1.5% over logit-only distillation. Per-class analysis shows the most significant improvements on texture-dependent categories like "vegetation" (+8.3%) and "building" (+6.1%), confirming that 2D priors effectively transfer texture information.
在 SemanticKITTI 上，2DPASS 達到 72.9% mIoU，超越先前最佳的純光達方法（Cylinder3D，67.8%）5.1%，甚至超越了數個在測試時使用相機的多模態方法。在 nuScenes 上，2DPASS 達到 79.4% mIoU，較先前最優提升 2.8%。消融研究展示了多尺度蒸餾相較單尺度貢獻了 +3.2% mIoU，特徵層級和 logit 層級蒸餾的組合比僅 logit 蒸餾提升了 +1.5%。逐類分析顯示在紋理依賴類別如「植被」（+8.3%）和「建築」（+6.1%）上改善最為顯著，確認了二維先驗有效地轉移了紋理資訊。

論證技巧 / 潛在漏洞紋理依賴類別的顯著改善是最有說服力的證據。但幾何依賴類別的改善幅度未被強調。

4. Conclusion — 結論

We have presented 2DPASS, a knowledge distillation framework that enables LiDAR-only networks to benefit from 2D image priors during training. By distilling multi-modal knowledge at multiple scales, 2DPASS achieves state-of-the-art performance on SemanticKITTI and nuScenes while requiring only LiDAR point clouds at inference time. This approach offers a practical solution for real-world deployment where sensor configurations may vary.
本文提出了 2DPASS，一種知識蒸餾框架，使純光達網路能在訓練時受益於二維影像先驗。透過多尺度的多模態知識蒸餾，2DPASS 在 SemanticKITTI 和 nuScenes 上達到最先進表現，同時在推論時僅需光達點雲。此方法為感測器配置可能變動的實際部署提供了務實的解決方案。

核心主張

透過訓練時的多尺度知識蒸餾，純光達網路可以獲得二維影像的紋理先驗，在部署時無需相機即可達到多模態方法的效能水準。

最強論點

逐類分析顯示紋理依賴類別（植被 +8.3%、建築 +6.1%）改善最大，精確地驗證了二維先驗轉移紋理資訊的核心假說。

最弱環節

知識蒸餾的效果受限於教師模型（融合分支）的品質，且訓練時仍需要配對的影像-點雲資料，限制了方法在缺乏配對資料場景中的應用。

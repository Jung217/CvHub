Abstract — 摘要

In this paper, we develop Position Embedding Transformation (PETR) for multi-view 3D object detection. PETR encodes the position information of 3D coordinates into image features, producing the 3D position-aware features. Object query can perceive the 3D position-aware features and perform end-to-end object detection. PETR achieves state-of-the-art performance on standard nuScenes dataset for both 3D object detection and BEV segmentation, surpassing previous methods by a significant margin of 3.6% mAP and 1.8% NDS compared to BEVDet. The framework is conceptually simple and does not require any view transformation or depth estimation module.

1. Introduction — 緒論

Multi-view 3D object detection from cameras is essential for autonomous driving. Existing approaches can be categorized into two paradigms: BEV-based methods that explicitly transform image features to BEV space using depth estimation, and query-based methods like DETR3D that use learnable 3D reference points. BEV-based methods require accurate depth estimation, which is error-prone, while DETR3D only queries sparse 3D reference points, limiting its ability to capture dense features. We propose PETR, which implicitly encodes 3D positional information into 2D image features through position embedding transformation, allowing object queries to directly interact with 3D-aware image features without explicit depth or BEV conversion.
基於相機的多視角三維物體偵測對自動駕駛至關重要。現有方法可分為兩個範式：使用深度估計將影像特徵顯式轉換至 BEV 空間的BEV 方法，以及使用可學習三維參考點的基於查詢的方法（如 DETR3D）。BEV 方法需要精確的深度估計（易出錯），而 DETR3D 僅查詢稀疏的三維參考點，限制了其捕捉密集特徵的能力。我們提出 PETR，透過位置嵌入轉換將三維位置資訊隱式編碼至二維影像特徵中，允許物體查詢直接與三維感知的影像特徵互動，無需顯式深度或 BEV 轉換。

邏輯角色以清晰的二分法框定現有方法，為 PETR 的「第三路線」建立動機。

2. Method — 方法

The core of PETR is the 3D Position Embedding (3D PE) module. For each pixel in the 2D image feature map, we first generate a ray from the camera origin through the pixel into 3D space. Along this ray, we sample D discrete depth values and obtain D 3D coordinates for each pixel. These 3D coordinates are then transformed into position embeddings through an MLP, which are added to the image features. This process converts the 2D image features into 3D position-aware features that implicitly encode depth information at every pixel. Object queries then use standard cross-attention to interact with these 3D-aware features for detection.
PETR 的核心是三維位置嵌入（3D PE）模組。對於二維影像特徵圖的每個像素，我們首先從相機原點穿過該像素生成一條射線至三維空間。沿此射線，我們採樣 D 個離散深度值並獲得每個像素 D 個三維座標。這些三維座標接著透過 MLP 轉換為位置嵌入，加入影像特徵中。此過程將二維影像特徵轉換為在每個像素隱式編碼深度資訊的三維位置感知特徵。物體查詢隨後使用標準交叉注意力與這些三維感知特徵互動以執行偵測。

3. Experiments — 實驗

We evaluate PETR on the nuScenes dataset. PETR with ResNet-101 backbone achieves 38.3% mAP and 44.2% NDS on the validation set, surpassing DETR3D by 3.0% mAP and BEVDet by 3.6% mAP. On the test set, PETR achieves 45.5% NDS. Ablation studies show that the 3D PE module is the critical component, contributing +12.1% mAP over the 2D baseline. The number of depth samples D=64 provides the best performance, with diminishing returns beyond this point. PETR also demonstrates strong robustness to camera extrinsic noise, outperforming BEV-based methods that rely on accurate camera calibration for depth estimation.
我們在 nuScenes 資料集上評估 PETR。使用 ResNet-101 骨幹的 PETR 在驗證集上達到 38.3% mAP 和 44.2% NDS，超越 DETR3D 3.0% mAP 和 BEVDet 3.6% mAP。在測試集上，PETR 達到 45.5% NDS。消融研究顯示三維 PE 模組是關鍵組件，相較二維基準貢獻了 +12.1% mAP。深度採樣數 D=64 提供最佳表現，超過此值後收益遞減。PETR 還展示了對相機外參噪音的強穩健性，超越依賴精確相機校正的 BEV 方法。

4. Conclusion — 結論

We have presented PETR, a simple and effective framework for multi-view 3D object detection that encodes 3D positional information directly into image features. Without requiring explicit depth estimation or BEV transformation, PETR achieves state-of-the-art performance on nuScenes while maintaining architectural simplicity. The position embedding transformation paradigm provides a clean and extensible approach to bridging 2D image features with 3D spatial reasoning.
本文提出了 PETR，一種簡潔有效的多視角三維物體偵測框架，將三維位置資訊直接編碼至影像特徵中。無需顯式深度估計或 BEV 轉換，PETR 在 nuScenes 上達到最優表現，同時維持架構簡潔性。位置嵌入轉換範式提供了一種簡潔且可擴展的方法來橋接二維影像特徵與三維空間推理。

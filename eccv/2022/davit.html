Abstract — 摘要

In this work, we introduce Dual Attention Vision Transformers (DaViT), a simple yet effective vision transformer architecture that is able to capture global context while maintaining computational efficiency. We propose to exploit self-attention mechanisms with both "spatial tokens" and "channel tokens" as two complementary forms of attention. With spatial tokens, the spatial dimension defines the token scope and the channel dimension defines the token feature dimension. With channel tokens, the channel dimension defines the token scope and the spatial dimension defines the token feature dimension. DaViT-Tiny, DaViT-Small, and DaViT-Base achieve 82.8%, 84.2%, and 84.6% top-1 accuracy on ImageNet-1K with 28.3M, 49.7M, and 87.9M parameters respectively.
本文介紹雙注意力視覺 Transformer（DaViT），一種簡潔且有效的視覺 Transformer 架構，能在維持計算效率的同時捕捉全域上下文。我們提出利用「空間標記」和「通道標記」作為兩種互補形式的自注意力機制。使用空間標記時，空間維度定義標記範圍，通道維度定義標記特徵維度。使用通道標記時，通道維度定義標記範圍，空間維度定義標記特徵維度。DaViT-Tiny、DaViT-Small 和 DaViT-Base 在 ImageNet-1K 上分別達到 82.8%、84.2% 和 84.6% top-1 精確度，參數量分別為 28.3M、49.7M 和 87.9M。
段落功能全文總覽——以空間/通道雙注意力作為核心創新。
邏輯角色摘要巧妙地利用「空間標記」和「通道標記」的對稱性建立了直覺理解。
論證技巧 / 潛在漏洞以對稱的定義方式描述兩種注意力，讓讀者容易理解其互補性。但「通道標記」的概念在直覺上不如空間標記明確。

1. Introduction — 緒論

Vision Transformers typically compute self-attention over spatial tokens, where each token represents a spatial patch and attention captures spatial relationships. However, this approach treats all channels equally and does not explicitly model inter-channel dependencies. Conversely, channel attention mechanisms like SE-Net capture channel-wise dependencies but lose spatial information. We observe that spatial and channel attention capture fundamentally different types of information and should be combined for comprehensive feature learning. DaViT alternates between spatial window attention and channel group attention within each block.
視覺 Transformer 通常在空間標記上計算自注意力，每個標記表示一個空間區塊，注意力捕捉空間關係。然而，此方法對所有通道一視同仁，未明確建模通道間的依賴性。反之，通道注意力機制如 SE-Net 捕捉通道方向的依賴性但損失了空間資訊。我們觀察到空間和通道注意力捕捉根本不同類型的資訊，應結合以實現全面的特徵學習。DaViT 在每個區塊內交替使用空間窗口注意力和通道分組注意力。
段落功能建立問題意識——空間注意力與通道注意力各有不足。
邏輯角色以「空間 vs. 通道」的二元分析框架建立雙注意力的動機。
論證技巧 / 潛在漏洞SE-Net 的引用增強了通道注意力的可信度，但 SE-Net 已是較老的方法，更近期的通道建模方法可能更具競爭力。

2. Method — 方法

DaViT consists of alternating spatial window attention and channel group attention layers. In spatial window attention, we partition the feature map into non-overlapping windows of size P x P and compute multi-head self-attention within each window, identical to Swin Transformer. In channel group attention, we transpose the feature dimensions: grouping channels into G groups and treating each spatial position as a token within each channel group. Self-attention is then computed across all spatial positions within each channel group, providing global spatial information exchange with O(G * HW * (C/G)^2) complexity, which is linear in HW when C/G is small.
DaViT 由交替的空間窗口注意力和通道分組注意力層組成。在空間窗口注意力中，我們將特徵圖分割為 P x P 大小的不重疊窗口，在每個窗口內計算多頭自注意力，與 Swin Transformer 相同。在通道分組注意力中，我們轉置特徵維度：將通道分為 G 組，並將每個空間位置視為各通道組內的標記。自注意力在每個通道組的所有空間位置上計算，提供O(G * HW * (C/G)^2) 複雜度的全域空間資訊交換，當 C/G 較小時對 HW 為線性。
段落功能核心方法——雙注意力的具體機制。
邏輯角色以「轉置維度」的操作巧妙地將通道注意力轉化為空間位置之間的注意力，概念優雅。
論證技巧 / 潛在漏洞複雜度分析清楚地展示了計算可行性。但通道分組可能破壞跨組的通道相關性。

3. Experiments — 實驗

We evaluate DaViT on ImageNet-1K, COCO detection, and ADE20K segmentation. DaViT-Base achieves 84.6% top-1 on ImageNet with 87.9M parameters, outperforming Swin-B (83.5%) and CSWin-B (84.2%) at comparable model sizes. On COCO detection with Mask R-CNN, DaViT-Small achieves 49.5 box AP, surpassing Swin-S by 1.4 AP. On ADE20K semantic segmentation with UperNet, DaViT-Base achieves 49.4 mIoU, outperforming Swin-B by 1.2 mIoU. Ablation studies confirm that channel attention contributes +0.7% ImageNet accuracy and +0.9 COCO AP over spatial-only baselines.
我們在 ImageNet-1K、COCO 偵測和 ADE20K 分割上評估 DaViT。DaViT-Base 以 87.9M 參數在 ImageNet 上達到 84.6% top-1，在相近模型大小下超越 Swin-B（83.5%）和 CSWin-B（84.2%）。在 COCO 偵測（Mask R-CNN）上，DaViT-Small 達到 49.5 box AP，超越 Swin-S 1.4 AP。在 ADE20K 語義分割（UperNet）上，DaViT-Base 達到 49.4 mIoU，超越 Swin-B 1.2 mIoU。消融研究確認通道注意力相較於僅空間的基準貢獻了 +0.7% ImageNet 精確度和 +0.9 COCO AP。
段落功能核心實驗——三大基準的全面比較。
邏輯角色消融實驗精確地量化了通道注意力的增量貢獻。
論證技巧 / 潛在漏洞與 Swin 的直接比較以相同框架（Mask R-CNN、UperNet）確保了公平性。但通道注意力的增量較小，其必要性可能被質疑。

4. Conclusion — 結論

We have presented DaViT, a dual attention vision transformer that combines spatial and channel attention for comprehensive feature learning. By alternating between window-based spatial attention and group-based channel attention, DaViT captures both local spatial patterns and global channel dependencies efficiently. Our extensive experiments demonstrate consistent improvements over strong baselines across classification, detection, and segmentation tasks.
本文提出了 DaViT，一種結合空間和通道注意力以實現全面特徵學習的雙注意力視覺 Transformer。透過交替使用基於窗口的空間注意力和基於分組的通道注意力，DaViT 高效地捕捉局部空間模式和全域通道依賴。廣泛的實驗展示了在分類、偵測和分割任務上相較於強基準的一致改善。
段落功能全文總結——重申雙注意力的互補性。
邏輯角色以「一致改善」的措辭強調方法的穩健性。
論證技巧 / 潛在漏洞結論簡潔有力，但未討論雙注意力帶來的額外計算成本。

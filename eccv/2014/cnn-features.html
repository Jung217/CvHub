Abstract — 摘要

Recent results indicate that the generic descriptors extracted from convolutional neural networks are very powerful. This paper adds to the evidence by evaluating CNN features on a diverse set of recognition tasks, including scene recognition, fine-grained recognition, attribute detection, and image retrieval. In most cases, the off-the-shelf CNN features — without any fine-tuning — outperform highly tuned state-of-the-art systems that have been custom-designed for each specific task. This result is astounding and suggests that the features learned from large-scale image classification transfer remarkably well.
近期結果表明，從摺積神經網路提取的通用描述子非常強大。本文透過在多樣化的辨識任務上評估 CNN 特徵，為此提供了更多證據，包括場景辨識、細粒度辨識、屬性偵測和影像檢索。在大多數情況下，開箱即用的 CNN 特徵——無需任何微調——超越了針對每個特定任務精心設計的最先進系統。此結果令人驚嘆，表明從大規模影像分類中學到的特徵具有極佳的遷移能力。

1. Introduction — 緒論

The success of Krizhevsky et al.'s CNN on ImageNet classification has raised the question of whether the internal representations learned by CNNs are useful beyond the specific task they were trained for. Conventional wisdom holds that features should be designed or learned specifically for each task. We challenge this view by showing that a CNN trained on ImageNet can serve as an excellent generic feature extractor for a wide range of visual recognition problems.
Krizhevsky 等人的摺積神經網路在 ImageNet 分類上的成功引出了一個問題：摺積神經網路學到的內部表示是否在其訓練任務之外仍有用處。傳統觀點認為特徵應針對每個任務專門設計或學習。我們挑戰此觀點，展示在 ImageNet 上訓練的摺積神經網路可以作為各種視覺辨識問題的優秀通用特徵提取器。

2. Method — 方法

Our approach is deliberately simple. We use a CNN pre-trained on ImageNet (specifically, the OverFeat network) as a fixed feature extractor. For each image, we extract the activation of the penultimate fully-connected layer (fc7) as a 4096-dimensional feature vector. This feature vector is then used with a simple linear SVM or nearest-neighbor classifier for the target task. No fine-tuning or domain adaptation is performed.
我們的方法刻意保持簡單。我們使用在 ImageNet 上預訓練的摺積神經網路（具體為 OverFeat 網路）作為固定的特徵提取器。對每張影像，我們提取倒數第二個全連接層（fc7）的激活作為 4096 維特徵向量。此特徵向量然後搭配簡單的線性 SVM 或最近鄰分類器用於目標任務。不進行微調或領域適應。

3. Cross-Task Evaluation — 跨任務評估

We evaluate CNN features on a comprehensive set of recognition tasks: (1) Object classification on PASCAL VOC 2007; (2) Scene recognition on MIT Indoor-67 and SUN-397; (3) Fine-grained recognition on CUB-200-2011 birds; (4) Attribute prediction on the LFWA faces dataset; (5) Image retrieval on Oxford Buildings and Paris Buildings. This diverse set of tasks spans different visual domains and difficulty levels, providing a thorough test of feature transferability.
我們在一組全面的辨識任務上評估 CNN 特徵：(1) PASCAL VOC 2007 上的物件分類；(2) MIT Indoor-67 和 SUN-397 上的場景辨識；(3) CUB-200-2011 鳥類上的細粒度辨識；(4) LFWA 人臉資料集上的屬性預測；(5) Oxford Buildings 和 Paris Buildings 上的影像檢索。這組多樣化的任務涵蓋不同的視覺領域與難度層級，為特徵可遷移性提供了徹底的測試。

4. Experiments — 實驗

Results are remarkable. On MIT Indoor-67, CNN features achieve 58.4% accuracy, surpassing the previous best of 51.4% which used hand-crafted features with spatial layout. On PASCAL VOC 2007, the CNN-based approach achieves 73.9% mAP, competitive with top methods. On CUB-200 fine-grained recognition, CNN features reach 51.2% without part annotations, outperforming some methods that use parts. These results consistently demonstrate that CNN features from ImageNet transfer extremely well to diverse recognition tasks.
結果令人矚目。在 MIT Indoor-67 上，CNN 特徵達到 58.4% 精度，超越先前使用具空間布局的手工設計特徵所達到的 51.4% 最佳成績。在 PASCAL VOC 2007 上，基於 CNN 的方法達到 73.9% mAP，與頂級方法具競爭力。在 CUB-200 細粒度辨識上，CNN 特徵在不使用部件標記的情況下達到 51.2%，優於一些使用部件的方法。這些結果一致證明了來自 ImageNet 的 CNN 特徵能極佳地遷移至多樣化的辨識任務。

5. Conclusion — 結論

We have demonstrated that CNN features trained on ImageNet provide an astounding baseline for a wide variety of visual recognition tasks. Without any fine-tuning or task-specific adaptation, these off-the-shelf features outperform or match highly specialized systems. This finding has profound implications for the field: it suggests that large-scale supervised pre-training produces representations that capture the fundamental structure of visual information.
我們已證明在 ImageNet 上訓練的 CNN 特徵為各種視覺辨識任務提供了令人驚嘆的基線。無需任何微調或任務特定適應，這些開箱即用的特徵超越或匹配高度專業化的系統。此發現對領域有深遠影響：它表明大規模監督式預訓練能產生捕捉視覺資訊基本結構的表示。
